# README

A test website.  

# Pre-requisites

* ruby 3.0.2p

* node v17.1.0

#Installation help:

```
$ brew install rbenv

# Update your bash_profile:
#
#  #rbenv
#  if command -v rbenv 1>/dev/null 2>&1; then
#    eval "$(rbenv init -)"
#  fi
#
#

$ rbenv install 3.0.2
$ rbenv global 3.0.2
$ brew install nodenv
$ nodenv install 17.1.0
$ nodenv global 17.1.0
$ corepack enable
```
